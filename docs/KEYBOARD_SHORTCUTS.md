# Keyboard Shortcuts

CodePath AI supports comprehensive keyboard navigation to ensure accessibility for all users.

## Global Shortcuts

| Shortcut | Action | Context |
|----------|--------|---------|
| `Tab` | Navigate to next interactive element | All pages |
| `Shift + Tab` | Navigate to previous interactive element | All pages |
| `Enter` | Activate focused button or link | All pages |
| `Space` | Activate focused button or checkbox | All pages |
| `Escape` | Close modal, dialog, or dropdown | When modal/dialog is open |
| `Alt + 1` | Navigate to Dashboard | Authenticated pages |
| `Alt + 2` | Navigate to Roadmap | Authenticated pages |
| `Alt + 3` | Navigate to Progress | Authenticated pages |

## Code Editor Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + S` (Windows/Linux) or `Cmd + S` (Mac) | Save code |
| `Ctrl + Enter` (Windows/Linux) or `Cmd + Enter` (Mac) | Run code |
| `Ctrl + Z` (Windows/Linux) or `Cmd + Z` (Mac) | Undo |
| `Ctrl + Y` (Windows/Linux) or `Cmd + Shift + Z` (Mac) | Redo |
| `Ctrl + F` (Windows/Linux) or `Cmd + F` (Mac) | Find |
| `Ctrl + H` (Windows/Linux) or `Cmd + H` (Mac) | Find and replace |
| `Ctrl + /` (Windows/Linux) or `Cmd + /` (Mac) | Toggle comment |

## Chat Interface Shortcuts

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift + Enter` | New line in message |
| `Escape` | Clear input or close chat |

## Lesson Navigation

## Form Navigation

| Shortcut | Action |
|----------|--------|
| `Tab` | Move to next field |
| `Shift + Tab` | Move to previous field |
| `Enter` | Submit form (when on submit button) |
| `Escape` | Cancel form or clear errors |

## List Navigation

| Shortcut | Action | Context |
|----------|--------|---------|
| `↑` / `↓` | Navigate up/down in vertical lists | Roadmap, lesson list |
| `←` / `→` | Navigate left/right in horizontal lists | Project cards |
| `Home` | Jump to first item | Any list |

| `Enter` | Select/open focused item | Any list |

## Modal/Dialog Shortcuts

| Shortcut | Action |
|----------|--------|
| `Escape` | Close modal/dialog |
| `Tab` | Navigate within modal (focus trapped) |
| `Shift + Tab` | Navigate backward within modal |

## Skip Links

| Shortcut | Action |
|----------|--------|
| `Tab` (on page load) | Reveal skip navigation links |
| `Enter` (on skip link) | Jump to main content |

## Accessibility Features

### Focus Indicators
- All interactive elements display a visible focus indicator (blue outline)
- Focus indicators meet WCAG 2.1 contrast requirements (3:1 minimum)

### Tab Order
- Tab order follows logical reading order (left-to-right, top-to-bottom)
- Hidden elements are excluded from tab order
- Modal dialogs trap focus within the modal

### Screen Reader Support
- All interactive elements have descriptive labels
- Form fields have associated labels
- Error messages are announced
- Dynamic content updates are announced via ARIA live regions

## Browser Compatibility

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## Tips for Keyboard Navigation

1. **Start with Tab**: Press Tab when a page loads to see the focus indicator
2. **Use Skip Links**: Press Tab once on any page to access skip navigation links
3. **Explore with Arrow Keys**: Use arrow keys to navigate within lists and menus
4. **Escape to Exit**: Press Escape to close any modal, dialog, or dropdown
5. **Enter to Activate**: Press Enter on any focused button or link to activate it
